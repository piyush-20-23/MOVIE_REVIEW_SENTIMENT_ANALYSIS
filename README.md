# 🎬 Sentiment Analysis Web App

A machine learning-based web application that classifies movie reviews as **positive** or **negative** using Natural Language Processing (NLP) techniques and Logistic Regression.

---

## 🚀 Project Overview

This project builds an end-to-end sentiment analysis pipeline:
- Processes raw text data
- Trains a machine learning model
- Deploys the model via a web interface

Users can input a movie review and instantly receive a sentiment prediction.

---

## 🧠 Tech Stack

- **Python**
- **Pandas** – Data handling  
- **NLTK** – Text preprocessing  
- **Scikit-learn** – Model building  
- **TF-IDF Vectorizer** – Feature extraction  
- **Logistic Regression** – Classification  
- **Matplotlib / WordCloud** – Visualization  
- **Streamlit** – Web application
